Translates name of season to 16 languages and scripts.

## Usage

```php
usee peterkahl\SeasonInfo\SeasonInfo;

echo SeasonInfo::getName('autumn', 'en-us'); # fall

echo SeasonInfo::getName('spring', 'it');    # primavera

```
